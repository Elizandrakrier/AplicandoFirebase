# AplicandoFirebase

## 🚀 Sobre o Projeto

O **AplicandoFirebase** é um projeto Android desenvolvido em Kotlin que demonstra a integração básica com o Firebase para inserção e gerenciamento de dados. O aplicativo apresenta uma tela de login onde os dados de acesso do usuário são inseridos manualmente para permitir a entrada no aplicativo. Após o login, o usuário é direcionado para uma tela de cadastro, onde pode inserir seu nome e idade. Os dados inseridos são registrados e acompanhados no Firebase, conforme a estrutura de nó definida.

## 📚 Funcionalidades

- **Tela de Login**: Autenticação manual do usuário para acesso ao aplicativo.
- **Tela de Cadastro**: Formulário para inserção de nome e idade do usuário.
- **Integração com Firebase**: Armazenamento e acompanhamento dos dados cadastrados diretamente no Firebase.

## 🛠 Tecnologias Utilizadas

- **Linguagem**: Kotlin
- **Plataforma**: Android
- **Serviço em Nuvem**: Firebase
